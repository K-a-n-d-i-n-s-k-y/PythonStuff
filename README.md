# PythonStuff
Stuff I made on borrowed time back in the days

Most of those were made on my Numworks calculator during school hours.

You might find some interesting stuff in there maybe,
some experimental project in there might become something if I ever get back to those.

Don't judge my optimisation, most of those had to run on [Numworks hardware](https://www.numworks.com/engineering/specs/) with a whopping unstable 45hz refresh rate

Nonetheless I'm still impressed by what's possible with little imagination and little computing power on these.


# Running the "Stuff"
Most of those import librairies like `kandinsky` that are proper to Numworks calculator, you might need to [run them on their emulator](https://www.numworks.com/simulator/) or on your own.
